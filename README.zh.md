# 🖼️ IMAGE-CUT

快速、私密且基于浏览器的图片编辑器。

> 无限免费 + 开源项目  
> 无需注册 · 无广告 · 无水印 · 无数据收集

[English](./README.md) | [한국어](./README.ko.md) | [简体中文](./README.zh.md) | [日本語](./README.ja.md)

IMAGE-CUT 是一款高性能的图片编辑工具，可直接在浏览器中运行，无需上传到服务器。基于 Canvas API 和 WebAssembly，所有图片处理都在您的设备本地完成，确保快速、安全的编辑，无数据泄露。

---

## ✨ 核心功能

*   **⚡ 无服务器处理**: 不向服务器上传图片。所有处理均在浏览器内完成。
*   **📐 精确调整大小**: 像素级精确调整图片尺寸。
*   **🎨 即时滤镜**: 实时应用灰度、复古和反转滤镜。
*   **🔄 格式转换**: 将编辑后的图片导出为 PNG、JPEG、WebP 或 GIF。
*   **🛡️ 隐私至上**: 您的数据永远不会离开您的计算机。100% 隐私和安全。

---

## 🚀 开始使用

1.  **上传**: 拖放图片或点击上传区域。
2.  **编辑**: 使用侧边栏调整大小、应用滤镜或更改格式。
3.  **下载**: 点击下载按钮，将编辑后的图片保存到本地。

---

## 🛠️ 技术栈

*   **Frontend**: Vanilla HTML5, CSS3, ES6+ JavaScript.
*   **Engine**: Canvas API & WebAssembly.
*   **Deployment**: 针对 GitHub Pages 进行优化，支持 `coi-serviceworker`。
