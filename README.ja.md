# 🖼️ IMAGE-EDITOR

高速、プライベートなブラウザベースの画像エディター。

> 完全無料 + オープンソース  
> 登録不要 · 広告なし · ウォーターマークなし · データ収集なし

[English](./README.md) | [한국어](./README.ko.md) | [简体中文](./README.zh.md) | [日本語](./README.ja.md)


IMAGE-EDITOR は、サーバーへのアップロードなしでブラウザ上で即座に動作する高性能な画像編集ツールです。Canvas API と WebAssembly を活用し、すべての画像処理をデバイス内でローカルに実行するため、データの漏洩を心配することなく高速かつ安全に編集できます。

---

## ✨ 主な機能

*   **⚡ サーバーレス処理**: サーバーに画像をアップロードしません。すべての処理はブラウザ内で行われます。
*   **📐 精密なサイズ変更**: ピクセル単位で正確に画像の寸法を調整できます。
*   **🎨 リアルタイムフィルター**: グレースケール、セピア、反転フィルターをリアルタイムで適用。
*   **🔄 フォーマット変換**: 編集した画像を PNG, JPEG, WebP, または GIF として書き出し。
*   **🛡️ プライバシー保護**: データがコンピュータの外に出ることはありません。100% のプライバシーとセキュリティ。

---

## 🚀 使い方

1.  **アップロード**: 画像をドラッグ＆ドロップするか、アップロードエリアをクリック。
2.  **編集**: サイドバーを使ってサイズ変更、フィルターの適用、またはフォーマットの変更。
3.  **ダウンロード**: ダウンロードボタンをクリックして、編集した画像をローカルに保存。

---

## 🛠️ 技術スタック

*   **Frontend**: Vanilla HTML5, CSS3, ES6+ JavaScript.
*   **Engine**: Canvas API & WebAssembly.
*   **Deployment**: `coi-serviceworker` により GitHub Pages に最適化.
