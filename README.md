<a name="english"></a>
## English

**Unlimited Free + Open Source**  
**No Signup · No Ads · No Watermarks · No Data Collection**

Image Cut is a simple and powerful image editor that runs entirely in your browser. Your images never leave your computer, ensuring 100% privacy.

### Key Features
- **Format Support**: Export to PNG, JPG, WebP, and GIF.
- **Resize**: Easily adjust image dimensions.
- **Filters**: Apply Grayscale, Sepia, and Invert filters.
- **Privacy First**: All processing is done locally via Canvas API and WebAssembly.
- **Deployment Ready**: Optimized for GitHub Pages with COI support.

### Tech Stack
- Vanilla HTML5, CSS3, and JavaScript (ES6+).
- Canvas API for image manipulation.
- `coi-serviceworker` for SharedArrayBuffer support.

## 한국어

**무제한 무료 + 오픈 소스**  
**가입 불필요 · 광고 없음 · 워터마크 없음 · 데이터 수집 없음**

이미지 컷(Image Cut)은 브라우저에서 완전히 실행되는 간단하고 강력한 이미지 에디터입니다. 이미지가 사용자의 컴퓨터를 떠나지 않으므로 100% 프라이버시가 보장됩니다.

### 주요 기능
- **다양한 포맷**: PNG, JPG, WebP, GIF 내보내기 지원.
- **크기 조절**: 이미지 해상도를 간편하게 조정.
- **필터**: 그레이스케일, 세피아, 반전 필터 적용.
- **개인정보 보호**: Canvas API와 WebAssembly를 통해 모든 처리가 로컬에서 수행됨.
