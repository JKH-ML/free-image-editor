# 🖼️ IMAGE-EDITOR

Fast, Private, and Browser-based Image Editor.

> Unlimited Free + Open Source  
> No Signup · No Ads · No Watermarks · No Data Collection

[English](./README.md) | [한국어](./README.ko.md) | [简体中文](./README.zh.md) | [日本語](./README.ja.md)

IMAGE-EDITOR is a high-performance image editing tool that runs instantly in the browser without server uploads. Powered by the Canvas API and WebAssembly, all image processing happens locally on your device, ensuring fast and secure editing without data leakage.

---

## ✨ KEY FEATURES

*   **⚡ Serverless Processing**: No image uploads to servers. Processing happens entirely within your browser.
*   **📐 Precise Resizing**: Adjust image dimensions with pixel-perfect accuracy.
*   **🎨 Instant Filters**: Apply Grayscale, Sepia, and Invert filters in real-time.
*   **🔄 Format Conversion**: Export your edited images to PNG, JPEG, WebP, or GIF.
*   **🛡️ Privacy First**: Your data never leaves your computer. 100% private and secure.

---

## 🚀 GETTING STARTED

1.  **Upload**: Drag and drop your image or click the upload area.
2.  **Edit**: Use the sidebar to resize, apply filters, or change formats.
3.  **Download**: Click the download button to save your edited image locally.

---

## 🛠️ TECH STACK

*   **Frontend**: Vanilla HTML5, CSS3, ES6+ JavaScript.
*   **Engine**: Canvas API & WebAssembly.
*   **Deployment**: Optimized for GitHub Pages with `coi-serviceworker`.
